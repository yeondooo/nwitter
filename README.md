# 트위터 클론코딩
해당 프로젝트는 노마드코더의 트위터 클론코딩 강의입니다.<br>
`React`와 `Firebase`를 사용했습니다.

## 🏁 프로젝트 실행 방법

1. 의존성 패키지를 설치합니다.

```
$ npm install
```

2. 프로젝트를 실행합니다.

```
$ npm start
```
